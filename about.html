About Run Tracker

A client-side dashboard for analyzing your Slay the Spire 2 runs. Everything runs in your browser — no server, no uploads, no accounts.

🚀 Getting Started

  1. Open Runs or Stats in Chrome or Edge.
  2. Click Load Folder and select your run history folder, which is typically here:
    %appdata%\SlayTheSpire2\steam\<YourSteamID>\profile1\saves\history\
  3. Your data persists for the session until you close the tab.
  4. The active run file is automatically skipped when a game is in progress.

📄 Pages

Runs

Browse individual runs with HP/gold charts, final deck, relics, potions, and a floor-by-floor timeline with full reward tracking. Hover any item for detailed tooltips.

Stats

Aggregate analytics across all runs: win rates, streaks, character performance, causes of death, HP analysis, plus sortable relic and card tables with synergy filtering.

Cards

Browse all 576 cards with images, costs, types, rarities, descriptions, and keyword badges. Filter by type, rarity, or character.

Relics

Browse all 288 relics with images, rarity/pool badges, and descriptions. Filter by rarity or character pool.

Potions

Browse all 63 potions with images, rarity/pool badges, and descriptions. Filter by rarity or character pool.

About

You are here! How to use the app, browser support, privacy info, and known issues.

🎮 Multiplayer Support

  • Automatically detects your Steam ID (the ID appearing most frequently across co-op runs).
  • All stats are scoped to your character only, even in multiplayer.
  • Filter by Solo / Co-op / All on both the Runs and Stats pages.
  • A "YOU" tag marks your character in co-op run displays and player tabs.

🌐 Browser Support

Chrome / Edge / Opera Full support Folder loading and all features work.
Firefox / Safari Not supported The Load Folder button does not work due to lack of webkitdirectory support. Workaround TBD.

🔒 Privacy

All data stays in your browser. Run files are read locally via the File API and never uploaded anywhere. Session data is stored in IndexedDB and automatically cleared when you close your browser tabs.

⚠ Known Issues

Deaths by Act — Event classification is not fully accurate. Some deaths during event-triggered combat may be miscategorized.

Regent card star costs are not included in the card encyclopedia data. The energy cost displays correctly but the secondary star cost is missing.

Abandoned runs do not count toward your win rate (they are excluded from the W/L calculation), but they do interrupt win streaks.

Card Upgrades in general are not reflected in the Final Deck view, nor is that data available in the Cards page. Fix TBD.

📚 Data Sources

  • Run files — .run JSON files generated by Slay the Spire 2 (schema version 8, game builds v0.98–v0.99+).
  • Card, relic, and potion data & images — Scraped from spire-codex.com.
